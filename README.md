# -digital-literacy-project

This repository contains all tasks completed as part of the **Digital Literacy Project**. The project focuses on developing essential digital skills, understanding online platforms, practicing coding, and promoting safe and responsible internet usage.

---

## Project Structure

### Task 1: Digital Literacy Infographic
- Created a one-page infographic explaining:
  - Digital literacy
  - Useful digital tools
  - Safe internet practices
  - Professional online presence
- Tool used: Canva 

---

### Task 2: Professional Communication
- Created/updated profiles on:
  - GitHub  
  - LinkedIn  
  - Stack Overflow  

- **GitHub:**
  - Created a profile README repository
  - Included name, branch, year, and learning goals  

- **LinkedIn:**
  - Updated Education section with degree, institution, and expected graduation year  

- **Stack Overflow:**
  - Set up a profile for asking and answering technical questions  

---

### Task 3: Coding & Collaboration Platforms

#### Part A:
- Completed a beginner-level coding challenge on **HackerRank**
- Practised:
  - Basic problem solving
  - Input-output handling
  - Logic building

#### Part B:
- Created a **Digital Literacy Awareness Quiz** using Google Forms
- Referred to VITyarthi for question design

---

### Task 4: Digital Communication & Social Media
- Wrote formal emails:
  - Requesting assignment deadline extension
  - Applying for an internship opportunity
- Followed proper email etiquette and formatting

### Part B:
- Checklist of:
  - Responsible social media Do’s and Don’ts

---

### Task 5: Cybercrime Awareness

#### Part A:
- Case study on **Cyberbullying**
  - Definition
  - Process (step-by-step)
  - Target groups
  - Consequences

#### Part B:
- Reflection report on cybercrime research
  - Includes prevention tips
  - UPI and financial safety guidelines
  - Reporting channels:
    - National Cyber Crime Portal: https://cybercrime.gov.in  
    - Helpline: 1930
